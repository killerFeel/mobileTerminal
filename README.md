# mobile_app

#### 介绍 
```
1.基于Vue3开发 
2.使用基于AJAX封装的axios插件
3.使用UI框架Vant v3
4.基础tabBar导航切换已设置
5.自定义通用页面标题栏配置组件
6.使用nprogress对页面切换进度条的动画展示
7.对vuex进行拆分
```

```
npm install
```
